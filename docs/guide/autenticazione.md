# Autenticazione

## Panoramica

Beyond Titles non implementa un sistema di autenticazione utente. Non esistono account, login, sessioni server-side o meccanismi di accesso controllato.

## Accesso Pubblico

Tutte le funzionalità sono liberamente accessibili senza restrizioni:

- Survey Behavioural & Competency-Based
- System Activity Log
- Changelog

## Ruoli e Permessi

Non esistono ruoli utente nel sistema attuale. Tutti gli utenti hanno accesso completo a tutte le funzionalità.
